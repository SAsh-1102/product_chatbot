Product Chatbot

This chatbot answers user queries based on your product catalog JSON file using embeddings and a local LLM.

Features:
- Answer questions about your products/services intelligently.
- Guide users politely if they ask unrelated questions.
- Works with free local GPT4All model and Chroma vector database.

Usage:
1. Install dependencies.
2. Run the chatbot using:
   python chatbot_ai.py
3. Type questions about products. Type 'exit' to quit.
